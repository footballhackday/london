Data Sources

Here you'll find a number of data sources you may wish to utilize as you build your project.

Stats.com Data

You can access data from Stats.com in three ways:

  • Directly from the XML files (but don't worry)
  • Via a Football API backend app

Full details can be found below.

XML Files

You'll be told how you can get the XML files on the day. Even if you don't want to directly access the XML files you'll still need them.

Football API Backend

The Football API is a Ruby application built by X8 that lets you access data via a RESTful API and stream the data via Pusher so you can simulate building real-time applications from Stats.com Premier League data.
